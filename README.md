# Install

- **Tkinter**
  
  - **Linux**  
  
     Debian-based  
        
     ```
     sudo apt-get install python3-tk
     ```  
        
     Arch-based
     
     ```
     sudo pacman -S tk
     ```
     
     Fedora-based
     
     ```
     sudo dnf install python3-tkinter
     ```
     
     CentOS, Oracle Linux... 
     
     ```
     sudo yum install -y tkinter tk-devel
     ```
- **Dependences**
  
    ```
    pip install -r requirements.txt
    ```
